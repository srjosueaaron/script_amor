¿CÓMO UTILIZAR EL PROYECTO?
Opción Uno:  Simplemente ingresa aquí -> https://script-amor.srjosueaaron.com/
Opción Dos:  Descargar el archivo APK e instarlo en tu dispositivo Android
Opción Tres: Descargar el archivo ZIP desde GitHub

Aprovechando la ocasión cercana al 14 de Febrero les presento una 
versión modificada del proyecto donde le puedes pedir a alguien
donde se muestran dos botones uno de SÍ y otro de NO.
Al dar clic o colocarse sobre el botón de "NO" se moverá aleatoriamente
en la pantalla.

Funciones principales
-Al inicio pida el nombre de la persona y se imprime en pantalla
-El diseño de la interfaz fue basado en el meme del cartel con sonido de banda de fondo
-El proyecto es responsivo para que se pueda visualizar tanto en 
dispositivos moviles o en una computadora

Se utilizaron simplemente tres archivos uno de HTML, otro de CSS y el 
último de JavaScript para dar la funcionalidad.

Les recuerdo que el código mostrado no fue totalmente creado por mí 
solamente modificado para quedarse lo mejor de tres videos que 
encontre en YouTube sobre el tema. Eres libre de utilizar el código
para lo que necesites o modificar a tus necesidades especificas.
